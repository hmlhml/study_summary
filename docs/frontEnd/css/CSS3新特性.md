## 新增选择器
1、～选择器：`div ~ p`  div之后的所有同级p  
2、属选择器*$^：`[attribute^=value]` `[attribute$=value]` `[attribute*=value]`      
3、伪类选择器： `E:first-of-type` `E:empty` `E:checked` `E:not(selector)`   
4、伪对象选择：`E::selection`

2、阴影（box-shadow）

4、线性渐变（gradient）  
5、媒体查询
